# Knowledge Object Standard

Version: 0.0.1

---

## Definition

A Knowledge Object (KO) is the smallest independently identifiable unit of knowledge inside SKOS.

---

## Mandatory Fields

ID

Title

Description

Author

Source

Language

Creation Date

Version

Category

Tags

Relations

Evidence

Status

---

## Example

KO-000001

Title:
Introduction to SKOS

Category:
Architecture

Language:
English

Version:
0.0.1

Status:
Draft

---

## Rule

Every knowledge asset stored inside SKOS must become a Knowledge Object.
