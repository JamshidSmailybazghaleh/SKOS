# SKOS Naming Standard

Version: 0.0.1

## Repository

Repository names use:

lowercase-with-hyphen

Example:

skos

knowledge-registry

intake-engine

---

## Directories

Directories use:

lowercase

Examples:

constitution

standards

docs

archive

---

## Markdown Files

Identity documents use:

UPPERCASE

Examples:

README.md

MANIFEST.md

CHANGELOG.md

LICENSE

Other documents use:

UPPERCASE_WITH_UNDERSCORE

Example:

NAMING_STANDARD.md

VERSION_POLICY.md

---

## Source Files

Source files use:

kebab-case

Examples:

main.ts

knowledge-parser.ts

metadata-builder.ts

---

## Classes

Classes use:

PascalCase

Example:

IntakeEngine

KnowledgeRegistry

MetadataBuilder

---

## Functions

Functions use:

camelCase

Example:

parseDocument()

buildMetadata()

validateInput()

---

## Constants

Constants use:

UPPERCASE_WITH_UNDERSCORE

Example:

MAX_RECORD_SIZE

DEFAULT_LANGUAGE

---

## Version Policy

Build IDs:

BUILD-000001

BUILD-000002

...

Sprint IDs:

Sprint 01

Sprint 02

...
